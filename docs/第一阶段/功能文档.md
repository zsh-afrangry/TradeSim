# TradeSim 功能文档

> 用户视角：这个系统能做什么？

---

## 1. 回测实验室

系统的核心页面。用户填写参数后点击"运行"，即可对指定股票在历史区间内模拟交易，获取完整的绩效报告。

### 基础环境配置

| 参数 | 说明 |
|---|---|
| 股票代码 | A 股代码，如 `000400`、`600585` |
| 回测区间 | 起止日期，日期选择器一站到底 |
| 数据频率 | 日线 / 5分钟 / 1分钟，支持日内高频回测 |
| 初始本金 | 模拟起始资金，默认 10 万 |
| 佣金费率 | 双边交易摩擦成本，默认万分之二点五 |
| 滑点 | 模拟真实挂单的价格偏差，默认 0.01 元 |

### 策略选择与参数

目前支持一种策略：**网格交易（GRID_TRADING）**。

| 参数 | 说明 |
|---|---|
| 价格上下限 | 网格的价格区间 |
| 网格类型 | 等比（百分比间距）或等差（固定价差） |
| 网格间距 / 格数 | 等比模式填间距百分比；等差模式填格数 |
| 底仓比例 | 首日按此比例买入底仓，如 0.5 = 50% 资金建仓 |
| 每格交易额 | 每次触发网格买卖时投入的资金量 |

## 2. 结果可视化

回测完成后，右侧展示四轴联动图表：

| 图层 | 内容 |
|---|---|
| 第一轴 | 真实股价走势 + 网格价格刻度线 + 买卖点散点标记 |
| 第二轴 | 策略净值曲线 vs 同期标的涨跌基准线（对跑比较） |
| 第三轴 | 动态仓位利用率（持仓市值占总净值的比例） |
| 第四轴 | 回撤幅度曲线（当前净值距历史高点的跌幅） |

图表下方同步展示交易流水明细表，点击图表上的买卖点可高亮对应流水行，反向点击流水行也可定位图表位置。

### 核心绩效指标

| 指标 | 说明 |
|---|---|
| 总收益率 | 期末净值相对初始本金的涨跌幅 |
| 最大回撤 | 回测期内净值从高点到低点的最大跌幅 |
| 总交易次数 | 触发买入 + 卖出的总笔数 |
| 胜率 | 盈利卖出次数占已完成配对交易的比例 |

## 3. AI 诊断

点击"AI 研判"按钮，右侧抽屉弹出，大模型以流式打字方式输出 Markdown 格式的诊断报告。

报告内容包括：
- 本次回测的核心指标解读
- 收益或亏损的归因分析（区间错配、极值崩溃、摩擦磨损等）
- 参数优化建议

> 需要在后端 `.env` 中配置有效的 `LLM_API_KEY` 才能使用。

---

## 4. 收藏与排行榜

回测完成后，点击"收藏"将本次结果永久保存。

**收藏库（Dashboard）** 页面展示所有已保存记录，按总收益率倒序排列，每条记录显示：股票代码、策略名称、收益率、最大回撤、胜率、交易次数、保存时间。

---

## 5. 历史复盘

在收藏库点击任意一条记录，进入详情页，可完整还原当时的回测现场：

- 当时的入参配置（策略参数快照）
- 完整的四轴图表
- 最后 100 笔交易流水（含手续费、滑点明细）
